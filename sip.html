This is Home Brew

Home Brew was created in my senior year of college as my Student Innovation Project, or SIP for short. It is a mobile application that would help users find coffee and tea recipes that would best suit them based on their preferences.

The application was created using React Native on Snack Expo. The application was tested on both Android and iOS devices, and the code was uploaded to GitHub for easy access.

I presented Home Brew at the Fall 2023 SIP Showcase, where it received the award for Most Market Viable.
